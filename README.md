# FireWatch — Web Development
Global Solution / FIAP

## Descrição
Site do projeto FireWatch com interatividade em JavaScript puro.

## Funcionalidades implementadas
- Slideshow com 3 imagens navegáveis
- Formulários com validação para impedir envio de campos vazios
- Quiz com 10 perguntas sobre queimadas e o projeto FireWatch
- Troca de tema de cor da página (escuro, verde e claro)

## Tecnologias utilizadas
- HTML5
- CSS3
- JavaScript puro
